# honeybee-server
Personal Honeybee Server

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/ladybug-tools/honeybee-server)

## Requirements:

`pip install -r requiremnets.txt`
`redis-server`
`celery worker`
`python run.py`
or
`./run-local.sh`
